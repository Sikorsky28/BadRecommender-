# 🤖 Настройка Telegram бота для SOLOWAYS

## 📋 Описание

Telegram бот предоставляет тот же функционал, что и веб-приложение, но в удобном формате мессенджера. Пользователи могут проходить опрос и получать персональные рекомендации БАДов прямо в Telegram.

## 🚀 Создание бота

### 1. Создание бота через @BotFather

1. **Откройте Telegram** и найдите @BotFather
2. **Отправьте команду:** `/newbot`
3. **Введите имя бота:** `SOLOWAYS Рекомендации БАДов`
4. **Введите username:** `SolowaysRecommendationsBot` (должен заканчиваться на Bot)
5. **Получите токен** - сохраните его для настройки

### 2. Настройка команд бота

Отправьте @BotFather:
```
/setcommands
```

И добавьте команды:
```
start - Начать опрос для получения рекомендаций
help - Помощь по использованию бота
```

## ⚙️ Настройка переменных окружения

### В Render.com Dashboard:

1. **Перейдите в Environment Variables**
2. **Добавьте переменные:**
   ```
   TELEGRAM_BOT_USERNAME = SolowaysRecommendationsBot
   TELEGRAM_BOT_TOKEN = ваш_токен_от_botfather
   ```

### Локальная разработка:

Создайте файл `.env`:
```bash
TELEGRAM_BOT_USERNAME=SolowaysRecommendationsBot
TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
```

## 🔧 Функциональность бота

### Основные возможности:

1. **Приветствие** - знакомство с ботом
2. **Ввод имени** - персонализация
3. **Выбор темы** - 13 категорий здоровья
4. **Опрос** - 15 вопросов с кнопками
5. **Email** - получение рекомендаций
6. **Завершение** - подтверждение отправки

### Интерфейс:

- **Inline кнопки** для выбора тем и ответов
- **Текстовые сообщения** для ввода имени и email
- **Эмодзи** для визуального оформления
- **Прогресс** - показ текущего вопроса

## 🛠️ Техническая реализация

### Архитектура:

- **TelegramBotService** - основной сервис бота
- **TelegramUser** - модель пользователя
- **Session Management** - управление сессиями
- **State Machine** - управление состояниями

### Состояния пользователя:

1. `START` - начало работы
2. `NAME` - ввод имени
3. `TOPIC` - выбор темы
4. `QUESTIONS` - прохождение опроса
5. `EMAIL` - ввод email
6. `COMPLETE` - завершение
